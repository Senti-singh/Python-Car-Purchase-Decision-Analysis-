# Python-Car-Purchase-Decision-Analysis-

This project analyzes automotive data to help users make informed decisions about purchasing cars. It includes functionalities to filter cars based on various criteria and compute average prices by company.

Features:
1) Data Analysis: Utilizes pandas library to explore and analyze automotive data from a CSV file.
2) Decision Making: Determines whether a car is worth buying based on criteria such as wheel-base and length.
3) Average Price Calculation: Calculates the average price of cars for each company based on user-defined conditions.
